# Backtesting Engine (Python)

A modular backtesting framework to test trading strategies with transaction cost, slippage, and performance metrics.

## Features
- Moving Average Strategy
- Transaction Cost & Slippage
- Portfolio Tracking
- Performance Metrics (Return, Sharpe Ratio)

## Project Structure
- src/ → Core modules
- notebooks/ → Demo notebook
- data/ → Sample dataset

## How to Run

```bash
pip install -r requirements.txt
python main.py
